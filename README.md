# Python APIs + Data + Databases ⚙️🔌

This is Phase 2 of my learning journey. Phase 1 was all core Python — no libraries, no APIs, just the language itself. I wanted to actually understand the fundamentals before reaching for frameworks, so every project there was built with nothing but standard library and manual logic.

Now I'm moving into the next layer: building HTTP APIs with FastAPI, connecting to real databases, and working with structured data properly. These are the things that sit between "I can write Python" and "I can build something a real system depends on" — and I want to understand them properly, not just copy-paste from tutorials.

I'm learning all of this in public. The code here is what I actually built while figuring things out. Some of it took me way longer than it should have, and some of it I'd write differently now. That's fine — that's what learning looks like. I'm improving every week and this repo is the proof of that.
